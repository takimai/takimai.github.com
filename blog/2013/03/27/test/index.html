ruby -e “$(curl -fsSL https://raw.github.com/mxcl/homebrew/go)” brew doctor brew update brew install rbenv brew install ruby-build

1.9.3-p392に切り替え

$ rbenv global 1.9.3-p392

システムのデフォルトrubyに戻す

$ rbenv global system

特定のディレクトリ以下のRubyバージョンを切り替え

$ rbenv local 1.9.3-p392

ローカル設定を解除

$ rbenv local –unset

インストールしたrubyやgemのパスを通す

$ rbenv rehash

インストール済みのRuby一覧を見る

$ rbenv versions
