Markdown記法まとめ

Mar 29th, 2013

Markdown はHTMLのMarkupをもじっているそうだ。
Octopressの記事はMarkdown形式で生成されるので、そのままMarkdown記法を使って書けば、レイアウトを楽に調整することができる。
ということで、記事を書くときによく使うと思われるものをピックアップしたいと思う。
記事で使った記法についても、備忘録としてこの記事に掲載していく。
